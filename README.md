# Netflix Data Analysis

This project analyzes the Netflix titles dataset using Python, Pandas, and Matplotlib.

## Project Overview
The goal of this project is to explore Netflix content and gain basic insights such as:
- Movie vs TV Show distribution
- Content growth over the years
- Popular countries and genres

## Files
- netflix_analysis.ipynb : Jupyter Notebook containing the analysis
- netflix_titles.csv : Dataset used in the project

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Results
The analysis shows that Netflix content has increased significantly over the years, with movies being more common than TV shows.

## How to Run
1. Clone the repository or download the files
2. Open `netflix_analysis.ipynb` in Jupyter Notebook
3. Run the cells from top to bottom
